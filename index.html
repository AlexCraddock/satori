satori

Satori is an elegant and clean color scheme for Vim.

Download this project as a .zip file Download this project as a tar.gz file

Satori is an elegant and clean color scheme for Vim, both for white or black background terminals. This color scheme has been developed under Terminall.app on Mac OS X but it also looks great in Linux too.

Install details:

Download satori.vim

$ mkdir -p ~/.vim/colors/
$ cp satori.vim ~/.vim/colors/

Edit .vimrc and insert "colorscheme satori". Here is an example:

syntax on
colorscheme satori
set autoindent
set hlsearch
